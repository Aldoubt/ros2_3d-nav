# 常用命令

## 1. 基础环境

每开一个新终端，先执行：

```bash
cd /home/yangxuan/ros2_ws/src
source /opt/ros/humble/setup.bash
source /home/yangxuan/ros2_ws/src/install/setup.bash
```

如果当前终端还在 `conda` 环境里，建议先执行：

```bash
conda deactivate
```

## 2. 实车前准备

### 2.1 激活 CAN

```bash
sudo ip link set can0 up type can bitrate 500000
```

### 2.2 启动底盘驱动

```bash
cd /home/yangxuan/ros2_ws/src
source /opt/ros/humble/setup.bash
source /home/yangxuan/ros2_ws/src/install/setup.bash
ros2 launch yhs_can_control yhs_can_control.launch.py
```

### 2.3 启动 MID360

```bash
cd /home/yangxuan/ros2_ws/src
source /opt/ros/humble/setup.bash
source /home/yangxuan/ros2_ws/src/install/setup.bash
ros2 launch octo_planner mid360_driver.launch.py
```

检查雷达话题：

```bash
ros2 topic list | grep livox
ros2 topic info /livox/lidar -v
ros2 topic info /livox/imu -v
```

## 3. FAST-LIVO2 建图

用途：
- 实时里程计
- 采图并保存全局 `.pcd`

启动命令：

```bash
cd /home/yangxuan/ros2_ws/src
source /opt/ros/humble/setup.bash
source /home/yangxuan/ros2_ws/src/install/setup.bash

ros2 launch fast_livo mapping_mid360_lio.launch.py \
  params_file:=/home/yangxuan/ros2_ws/src/FAST-LIVO2-ROS2/config/mid360_lio_only.yaml \
  use_rviz:=true
```

说明：
- 这套配置会保存点云地图。
- 正常退出后会输出到：
  `/home/yangxuan/ros2_ws/src/FAST-LIVO2-ROS2/Log/PCD/all_raw_points.pcd`

建图时检查：

```bash
ros2 topic info /cloud_registered -v
ros2 topic info /aft_mapped_to_init -v
ros2 run tf2_ros tf2_echo odom livox_frame
```

## 4. FAST-LIVO2 重定位模式

用途：
- 只提供 `/cloud_registered`
- 只提供 `odom -> livox_frame`
- 不覆盖已有 `.pcd`

启动命令：

```bash
cd /home/yangxuan/ros2_ws/src
source /opt/ros/humble/setup.bash
source /home/yangxuan/ros2_ws/src/install/setup.bash

ros2 launch fast_livo mapping_mid360_lio.launch.py \
  params_file:=/home/yangxuan/ros2_ws/src/FAST-LIVO2-ROS2/config/mid360_lio_relocalization.yaml \
  use_rviz:=true
```

## 5. 点云重定位验证

用途：
- 验证 `map -> odom`
- 验证重定位是否能和 `FAST-LIVO2` 对齐

启动命令：

```bash
cd /home/yangxuan/ros2_ws/src
source /opt/ros/humble/setup.bash
source /home/yangxuan/ros2_ws/src/install/setup.bash

ros2 launch octo_planner mid360_relocalization.launch.py \
  map_path:=/home/yangxuan/ros2_ws/src/FAST-LIVO2-ROS2/Log/PCD/all_raw_points1.pcd \
  cloud_topic:=/cloud_registered \
  base_frame_id:=livox_frame \
  odom_frame_id:=odom \
  global_frame_id:=map
```

检查项：

```bash
ros2 topic info /initial_map -v
ros2 topic info /localization/pose_with_covariance -v
ros2 run tf2_ros tf2_echo map odom
ros2 run tf2_ros tf2_echo odom livox_frame
```

如果要手动给初始位姿，单独开一个 RViz：

```bash
cd /home/yangxuan/ros2_ws/src
source /opt/ros/humble/setup.bash
source /home/yangxuan/ros2_ws/src/install/setup.bash
rviz2
```

RViz 里建议：
- Fixed Frame 设为 `map`
- 添加 `/initial_map`
- 添加 `/cloud_registered`
- 使用 `2D Pose Estimate` 往 `/initialpose` 发初始位姿

## 6. PCD 转 OctoMap

### 6.1 直接导入 PCD，不开 GUI

```bash
cd /home/yangxuan/ros2_ws/src
source /opt/ros/humble/setup.bash
source /home/yangxuan/ros2_ws/src/install/setup.bash

ros2 launch jie_octomap import_pcd_map.launch.py \
  launch_import_gui:=false \
  pcd_file:=/home/yangxuan/ros2_ws/src/FAST-LIVO2-ROS2/Log/PCD/all_raw_points.pcd
```

### 6.2 启动 GUI 查看导图效果

```bash
cd /home/yangxuan/ros2_ws/src
source /opt/ros/humble/setup.bash
source /home/yangxuan/ros2_ws/src/install/setup.bash

ros2 launch jie_octomap import_pcd_map.launch.py \
  import_gui_python:=/home/yangxuan/ros2_ws/.venv-jie3d/bin/python
```

如果 GUI 依赖有问题，先检查：

```bash
/home/yangxuan/ros2_ws/.venv-jie3d/bin/python - <<'PY'
import numpy, scipy, sklearn, open3d, sys
print(sys.executable)
print("numpy:", numpy.__version__, numpy.__file__)
print("scipy:", scipy.__version__, scipy.__file__)
print("sklearn:", sklearn.__version__, sklearn.__file__)
print("open3d:", open3d.__version__, open3d.__file__)
PY
```

## 8. 常见检查

查看节点：

```bash
ros2 node list | sort
```

查看关键话题：

```bash
ros2 topic list | sort | grep -E '^/livox|^/cloud_registered|^/aft_mapped_to_init|^/initial_map|^/localization'
```

查看里程计 TF：

```bash
ros2 run tf2_ros tf2_echo odom livox_frame
```

查看重定位 TF：

```bash
ros2 run tf2_ros tf2_echo map odom
```
